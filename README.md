# To-Do List Application

A simple command-line application for managing tasks with a colorful, user-friendly interface.

## Features:
- Add new tasks with descriptions
- View all tasks with completion status
- Mark tasks as completed or active
- Delete tasks
- View task statistics
- Colorful terminal interface with visual feedback

## How to Use:
1. Run the application with `python simple_todo.py`
2. Navigate the menu using numbers 1-7
3. Add tasks, mark them as completed, or delete them as needed
4. View statistics to track your progress

## Requirements:
- Python 3.x

## Application Structure:
- `simple_todo.py`: Main application file with Task and TodoList classes
- Uses basic Python programming concepts, classes for data structures, and conditional statements

## Menu Options:
1. Add a new task
2. List all tasks
3. Mark task as completed
4. Mark task as active
5. Delete a task
6. Show task statistics
7. Exit

## UI Features:
- Colorful terminal interface
- Visual feedback with emojis
- Strikethrough formatting for completed tasks
- Clear screen between operations
- Informative prompts and messages
